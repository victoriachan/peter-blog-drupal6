To enable this module:

1. Upload it to your sites/all/modules folder and enable it on the admin/build/modules page.

2. Configure one of your WYSIWYG profiles to use the SyntaxHighlighter button.

3. Verify that TinyMCE can insert syntax highlighted code sections correctly when you create or edit
   a page using your WYSISWYG profile.
